Explanation of different files and folders in cronological order of crearion of the code: 

Data_exploration: file for the first assignment and about exploratory data analysis
Files for Feature engeneering: folder containing all the codes used for the features engineering of the variables inside the original datasets
First research question: codes used to find info to reply to the first research question (articles and candidates gender)
Seasonality: code used to test the differend windows of seasonality with new features
added_evaluation: code used to add evaluation to the model. This code have been made before the final one
Final models with evaluation and features with images: code of the last models tested using both evaluation, all the features and features extracted from the images

In each folder is possible to find another file with detailed info for each file in the folder
